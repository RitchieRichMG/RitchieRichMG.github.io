Less stress. More done.

TaskForge is a personal automation assistant for busy students and builders who manage school, work, and daily life. It helps organize tasks automatically, but you stay in control.

What TaskForge can do

  • Suggest calendar events from important emails
  • Turn messages into reminders
  • Organize schedules and to‑dos
  • Learn from past actions to improve suggestions

Safety rules (non‑negotiable)

  • No actions without permission
  • Risky actions require approval
  • Least‑access permissions for email/calendar
  • Every action is recorded in an audit log

Simple flow

1) Connect accounts
User logs in and links email + calendar.
2) TaskForge suggests
System shows suggested actions, not automatic actions.
3) You approve
Anything risky requires a clear approval click.
4) Workers execute
Background workers run longer tasks safely and quickly.
